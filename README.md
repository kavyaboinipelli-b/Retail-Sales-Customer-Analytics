# Retail Sales Customer Analytics

## Project Overview

Analyzed retail sales data using MySQL, Python, Power BI, and Excel to identify sales trends, profitable categories, top-performing states, and business opportunities.

## Tools Used

- MySQL
- Python
- Pandas
- Matplotlib
- Power BI
- Excel

## Key Analysis

1. Total Sales and Profit Analysis
2. Top States by Sales
3. Sales by Category
4. Profit by Category
5. Top Sub-Categories
6. Sales Ranking using Window Functions
7. Category Profit Analysis using CTE

## Key Insights

- Technology category generated the highest sales and profit.
- California was the top-performing state.
- Phones were the highest-selling sub-category.
- Furniture generated high sales but relatively low profit.
- High discounts contributed to losses in several cities.
